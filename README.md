
# CCSS Consultory Frontend

Welcome to the frontend application for the CCSS Consultory system. This system provides a comprehensive management interface for Medical Consultations, offering modules to manage Patients, Diseases, Drugs, and Users.

A core defining feature of this project is the **"Modo Atención" (Triage UI)**, a dynamic medical queue that sorts patients by the criticality/priority of their diagnosed conditions, automating the doctor's workflow by always presenting the most urgent patient first.

## Getting Started

### Prerequisites
Make sure you have Node installed (v18+).

### Installation
Run the following command to install required dependencies:
```bash
npm i
```

### Running the Code
Start the development server with Vite:
```bash
npm run dev
```
